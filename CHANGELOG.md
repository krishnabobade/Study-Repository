# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-04-27
### Added
- Enterprise-grade Authentication with JWT and Refresh Tokens.
- Dual-theme (Light/Dark) UI system using Tailwind CSS and CSS Variables.
- Advanced Document Preview and File Management.
- Secure Faculty Moderation capabilities.
- Integrated 3D Interactive Spline Robot on login page.
- Comprehensive SEO metadata and Core Web Vitals optimization.
- Legal Pages (Privacy Policy, Terms of Service, Cookie Consent).
- Strict Security Headers (Helmet, CORS) and Rate Limiting.
- Fully configured CI/CD ready deployment architecture.
